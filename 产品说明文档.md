# MindMatch — 产品说明文档

> 版本：v2.0 | 更新日期：2026-06-02

---

## 一、产品定位

**MindMatch** 是一款基于心理学量表的职业匹配工具。用户通过 4 个交互式"思维游戏"完成自我评估，系统据此生成个人思维肖像，并推荐最匹配的职业方向与具体岗位。

核心价值主张：**不是选职业，是发现你的思维方式天然适配什么。**

---

## 二、功能概览

### 2.1 四个思维游戏

| 游戏 | 心理学基础 | 交互方式 | 产出维度 |
|------|-----------|---------|---------|
| G1 核心驱动力 | McClelland 三需要理论 | 卡片点击（6 选 3 × 2 轮） | 成就需要、权力需要、亲和需要 |
| G2 职业锚 | Schein 职业锚理论 | 翻牌保留（3 阶段逐步淘汰） | 技术职能、管理能力、自主独立、安全稳定、创业创造、服务奉献、挑战纯粹、生活风格 |
| G3 认知风格 | Riding 认知风格理论 | 归类排序（先分两类再排序） | 整体—分析维度 |
| G4 意义建构 | 存在主义心理学 + Frankl 意义疗法 | 六帧场景双重反应 | 存在感、意义寻找 |

### 2.2 思维肖像

完成游戏后，系统生成两部分报告：

1. **维度概览**（4 张图表卡片）：雷达图/条形图展示各游戏维度得分
2. **肖像报告**：文字描述每个维度的表现及整体画像特征

### 2.3 AI 深度解读

基于思维肖像，调用 AI 大模型（腾讯混元）生成个性化深度分析，包括：
- 思维模式的核心特征
- 潜在优势与盲点
- 与不同职业环境的适配分析

> 需完成全部 4 个游戏后解锁。

### 2.4 职业匹配

6 大职业方向 × 每方向 9 个具体职业，两级匹配：

**第一级：方向匹配** — 6 个方向按匹配度排名：
- 系统建构者（架构/系统/技术战略）
- 深度解读者（研究/分析/洞察）
- 创意塑造者（设计/产品/创意）
- 人际联结者（人力/社交/赋能）
- 价值驱动者（公益/政策/社会创新）
- 赋能陪伴者（教育/培训/咨询）

**第二级：职业匹配** — Top 3 方向各展示 9 个具体职业，按匹配度排序。

> 需完成全部 4 个游戏后解锁。

## 三、匹配算法说明

### 3.1 核心思路

将用户的 14 维得分与每个职业方向的"理想画像"对比，越接近理想则匹配度越高。

### 3.2 高斯核匹配

使用高斯径向基函数（RBF）衡量用户得分与理想值的距离：

```
匹配度 = exp(-(用户得分 - 理想值)² / (2σ²))
```

- 完全匹配 → 1.0
- 差距 0.25 → ≈0.61
- 差距 0.5 → ≈0.14

相比旧版的阶跃函数（非 0 即 1），高斯核提供平滑渐变的匹配度，消除了"差一点就得 0 分"的不合理现象。

### 3.3 两阶段加权

**问题**：G2 有 8 个维度，G3 只有 1 个维度，直接加权导致 G2 碾压 G3。

**解法**：
1. **Phase 1**：游戏内归一化 — 每个游戏各维度加权平均 → 归一化到 [0,1]
2. **Phase 2**：跨游戏加权 — 用 `gameWeights` 对 4 个游戏分别赋权 → 最终 [0,100]

### 3.4 双引擎验证

- **主引擎**：线性加权法（高斯核 + 两阶段加权）
- **辅引擎**：TOPSIS 距离法（多属性决策）
- 两引擎结果独立计算，以线性法为主排序

## 四、使用流程

```
首页 → 选择游戏 → 完成游戏 → 结果页（肖像 + AI 解读 + 匹配入口）
                                    ↓
                         任意 1-3 个游戏：显示完成度提示
                         全部 4 个游戏：解锁 AI 解读 + 职业匹配
```

### 快速开始

1. 访问线上地址
2. 从首页进入任意游戏，按引导完成
3. 完成后自动跳转结果页
4. 完成 4 个游戏后，可查看 AI 深度解读和职业匹配

### 数据存储

所有数据存储在浏览器 localStorage 中：
- 游戏数据：`mindmatch_game1` ~ `mindmatch_game4`
- API Key（仅本地开发）：`mindmatch_api_keys`
- AI 解读缓存：`mindmatch_insight_*`

## 五、技术架构

### 5.1 技术栈

| 层 | 技术 |
|----|------|
| 前端 | 纯 HTML/CSS/JS + ES Modules，零构建 |
| 图表 | ECharts 5（CDN） |
| AI | 腾讯混元 API（hunyuan-lite） |
| 部署 | 腾讯云 CloudBase 静态托管 |
| API 代理 | CloudBase SCF 云函数 |
| 数据 | localStorage |
