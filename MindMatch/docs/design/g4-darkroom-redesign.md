# G4「六帧手记」暗房风格改造方案

## 问题诊断

G2 和 G4 视觉基调撞车：都在 hsl(28-38) 暖金/琥珀色域，暗底+暖光的模板重复。两个游戏仿佛同一主题的不同皮肤。

## 设计方向：摄影暗房

**核心隐喻**：六帧手记 → 六张在暗房中显影的照片。每个场景是一张正在药水中浮现的底片，两次反应选择是显影液→定影液的工序。

**与 G2 的区分**：
| | G2 | G4 |
|---|---|---|
| 色相 | hsl(28-38) 暖橙 | hsl(0-10) 深红 |
| 情绪 | 校园怀旧、温暖 | 私密暗房、沉浸 |
| 光线 | 夕阳散射光 | 单灯 safelight 集中光 |
| 核心意象 | 树影、传单、拍立得 | 药水、底片、显影盘 |

## 逐页设计

### 引导页 —「暗房门缝」

想象推开暗房门之前——门下渗出的红光，门上贴着冲洗流程标签。

- **背景**：极暗底 + 从页面中央偏上洒下的 safelight 红光（模拟头顶红灯）
- **标题**："六帧手记" — 红色调，衬线体，像暗房标签
- **描述文字**：淡暖灰，低对比
- **CTA 按钮**：暗房开关式按钮，左侧齿孔保留（呼应胶片），hover 时红光增强
- **装饰**：底部微微的红光"漏出"效果

### 场景卡片 —「显影盘」

每帧是一张正在显影盘中浮现的照片。场景卡片 = 显影盘的液体表面。

- **整体卡片**：
  - 暗红边框（模拟显影盘边缘）
  - 微弱的液体表面反光渐变
  - 进入动画：从暗到亮，"底片浮现"（brightness 0.3→1）
- **进度条**：改为一排红色小指示灯（暗房计时器），active 态亮红光，done 态暗红
- **场记标签**：暗房工序标签风格
- **标题**：红色衬线体，像写在底片袋上的字
- **故事正文**：淡灰暖白，低对比阅读

### 插图区 —「Safelight 下的底片」

图片不再用 grayscale(100%) + opacity:0.85，改为：

```css
filter: brightness(0.7) sepia(0.3) saturate(0.4) hue-rotate(-15deg);
border: 1px solid hsla(0, 30%, 25%, 0.5);
box-shadow: 0 0 20px hsla(0, 60%, 20%, 0.15);
```

效果：暗红暖色调，模拟在 safelight 红光下观看照片。图片"泛红"但不失细节。

### 反应卡片 —「药水工序」

两张反应卡片 = 两道药水工序：
1. **反应①（Presence）** — 「显影液」：让画面浮现，问"看到这段记忆，你的感受"——在帮你显影
2. **反应②（Search）** — 「定影液」：让画面固定，问"接下来你更想"——在帮你定影

视觉处理：
- 两张卡片用微弱的颜色区分（显影液略暖，定影液略冷一点）
- 卡片背景：暗红液体表面质感
- 卡片标题：药水标签风格
- 反应② 在反应① 完成后"浮现"（已有的 reveal 动画保留，加红光渐入）

### 选项按钮 —「镊子夹取」

- 默认：暗红半透明底，文字淡暖白
- hover：边框红光增强，底色微微提亮
- 选中：红色 safelight 高亮边框 + 微红光晕

### 完成页 —「晾干挂线」

六帧已经冲洗完毕，挂在暗房的晾干绳上。

- **中心卡片**：像一张冲洗完成、夹在绳上的照片
- **类型展示**：相纸背面手写的标签（emoji + label + title）
- **描述文字**：淡暖灰，阅读舒适
- **按钮**：暗红基调，"走出暗房"的暗示

## 动画设计

| 动画 | 效果 | 时机 |
|---|---|---|
| 底片浮现 | brightness 0.3→1 + 红光渐弱 | 每个场景加载 |
| 红灯呼吸 | safelight 微弱的亮度脉冲 | 全程背景 |
| 药水波纹 | 反应卡片微弱的 scale 呼吸 | 等待选择时 |
| 选项选中 | 红光边框脉冲 + 微光晕 | 点击选项 |
| 定影渐入 | 反应② 从暗到亮，red glow fade-in | 反应①完成后 |
| 晾干展示 | 卡片从上方滑入（像从绳子上取下） | 完成页 |

## 改动范围

### 只改 CSS + 少量 JS 文案

| 文件 | 改动 | 说明 |
|---|---|---|
| `games/game4-meaning-construction.html` | **重写全部内联 CSS** | 用新色板替换全部样式 |
| `js/games/strategies/dual-reaction.js` | **微调**（可选） | 标题文案可选改为"洗印中…"等暗房用语 |
| `js/games/game4-data.js` | 不改 | 数据结构不变 |
| `games/images/s1~s6.png` | 不改 | 图片保留，CSS 滤镜改渲染效果 |

### HTML 类名保持不変

所有 `g4-*` class 名称不变，只改 CSS 属性值。策略文件的 HTML 模板完全兼容。

## 预期效果

- G2（暖橙夕阳）和 G4（深红暗房）进入页面即能感受到明确区分
- 暗房主题与"六帧手记"的内在机制（翻看/反应/显影）概念自洽
- 沉浸感更强，适合 G4 较长的阅读体验
- 不改变任何数据结构和游戏逻辑
